# Project 1 Prompt History

This directory contains a curated prompt history for Project 1, the AI Marketing
Operations Agent.

This is not a raw chat export. It is a professional reconstruction of the
milestone prompts used to guide Codex/Claude-style AI coding agents through a
controlled implementation process. Each prompt captures the goal, scope,
constraints and expected verification for a meaningful project increment.

The purpose is to show how AI-assisted engineering was directed:

- deterministic automation before LLM interpretation;
- APIs before browser automation when an API exists;
- human approval before sensitive automated action;
- tests, typing, linting and documentation at each milestone;
- clear boundaries around secrets, generated files and runtime side effects.

Most entries are labeled as curated reconstructions because the exact historic
chat wording is not preserved in this repository. The reconstructions are based
on the implemented milestone, repository documentation and current source of
truth.
