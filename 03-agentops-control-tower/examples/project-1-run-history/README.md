# Project 1 Run History Evidence

Project 1 produces local workflow artifacts that Project 3 is expected to
observe in future milestones.

Planned sources include:

- run-history JSONL files
- approval requests JSONL files
- generated Markdown reports

These artifacts can support AgentOps questions such as:

- Which workflows ran?
- Which runs failed or produced warnings?
- Which recommendations required human approval?
- Which reports and artifacts were produced?

Milestone 1 does not parse or ingest these files.
