# Prompt History Index

Prompt histories are distributed by project. This index points to the canonical
location for each project's curated milestone prompts and implementation notes.

## Project Locations

- Project 1 prompt history is stored under
  `docs/prompt-history/project-1/`.
- Project 2 prompt history is stored under
  `02-agent-toolkit-mcp/docs/prompt-history/`.
- Project 3 prompt history is stored under
  `03-agentops-control-tower/docs/prompt-history/`.

Project 1 uses the original root-level prompt-history convention. Projects 2
and 3 use project-local `docs/prompt-history/` folders so their prompts,
verification notes and implementation history live with the project artifacts
they describe.
